Your company provides GitHub Copilot or Gemini, but you work better with Claude Code or OpenCode. Good news: they are compatible. This guide shows you how to connect them together, without breaking anything, without bending the rules.

Votre entreprise vous fournit GitHub Copilot ou Gemini, mais vous travaillez mieux avec Claude Code ou OpenCode. Bonne nouvelle : les deux sont compatibles. Ce guide vous montre comment les brancher ensemble, sans rien casser, sans contourner les règles.

Your company pays for GitHub Copilot or Gemini. You have no choice over the subscription, but the chat interface or autocompletion doesn't cover your needs. L'entreprise paye GitHub Copilot ou Gemini. Vous n'avez pas le choix de l'abonnement, mais l'interface chat ou l'autocompletion ne couvrent pas vos besoins.

❌ Loss of reflexes and speed Perte de réflexes et de vitesse

✅ A transparent local proxy Un proxy local, transparent

A small local server translates calls between your preferred tool and the enterprise subscription API. Nothing is installed on the company's servers. Un petit serveur local traduit les appels entre votre outil préféré et l'API de l'abonnement entreprise. Rien n'est installé sur les serveurs de l'entreprise.

✅ Standards compliant Conforme aux standards

Traffic goes through the official subscription. No personal API key, no parallel account. You stay within the framework authorized by your organization. Le trafic passe par l'abonnement officiel. Pas de clé API personnelle, pas de compte parallèle. Vous restez dans le cadre autorisé par votre organisation.
