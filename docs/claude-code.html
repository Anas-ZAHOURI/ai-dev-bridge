Quick StartQuick Start

3 commands to connect Claude Code to your Copilot or Gemini Code Assist license. No hack — just a local translator.3 commandes pour connecter Claude Code à votre licence Copilot ou Gemini Code Assist. Aucun hack — juste un traducteur local.

Warning: Copilot token consumption Avertissement : consommation de tokens Copilot

This method uses your GitHub Copilot token quota. Claude Code is extremely token-hungry: it consumes 10 to 100 times more tokens than a regular Copilot chat.Cette méthode utilise votre quota de tokens GitHub Copilot. Claude Code est extrêmement gourmand : il consomme 10 a 100 fois plus de tokens qu'un chat Copilot classique.

  • A single "edit this file" command = 50,000 to 150,000 tokensUne seule commande "edite ce fichier" = 50 000 a 150 000 tokens
  • A 30-minute session = the equivalent of a full day of Copilot ChatUne session de 30 minutes = l'equivalent d'une journee entiere de Copilot Chat
  • Multiple users report having exhausted their monthly quota in just a few daysPlusieurs utilisateurs rapportent avoir epuise leur quota mensuel en quelques jours

See the Token consumption section for details and optimization strategies.Voir la section Consommation de tokens pour les details et les strategies d'optimisation.

How it worksComment ça marche

Claude Code speaks the Anthropic Messages API format. GitHub Copilot speaks OpenAI format and Google Gemini speaks Generative Language format. A local proxy translates between them.Claude Code parle le format Anthropic Messages API. GitHub Copilot parle le format OpenAI et Google Gemini le format Generative Language. Un proxy local traduit entre eux.

Claude Code ───> Local Proxy ─┬─> GitHub Copilot API (terminal) (localhost:4141) │ (api.github.com) │ Anthropic Format Automatic └─> Google Gemini API POST /v1/messages Translation (generativelanguage.googleapis.com) The proxy also handles: Authentication (GitHub OAuth / Google Cloud) Automatic token renewal Removal of "thinking" parameter Rate limitingClaude Code ───> Proxy local ─┬─> API GitHub Copilot (terminal) (localhost:4141) │ (api.github.com) │ Format Anthropic Traduction └─> API Google Gemini POST /v1/messages automatique (generativelanguage.googleapis.com) Le proxy gère aussi : Authentification (OAuth GitHub / Google Cloud) Renouvellement automatique du token Suppression du paramètre "thinking" Rate limiting

Why does Claude Code consume so much?Pourquoi Claude Code consomme autant ?

Claude Code works differently from a simple chat:Claude Code fonctionne differemment d'un simple chat :

  • Massive context: for each request, Claude Code sends the contents of open files, conversation history, tool results, etc. A single message can contain 10,000 to 50,000 context tokens.Contexte massif : a chaque requête, Claude Code envoie le contenu des fichiers ouverts, l'historique de conversation, les resultats d'outils, etc. Un seul message peut contenir 10 000 a 50 000 tokens de contexte.
  • Multiple calls: a single user command can generate 3 to 15 cascading API calls (file reading, reasoning, editing, verification...).Appels multiples : une seule commande utilisateur peut generer 3 a 15 appels API en cascade (lecture de fichiers, reflexion, edition, verification...).
  • Background requests: without optimization flags, Claude Code makes additional calls for completions, suggestions and telemetry.Requetes en arrière-plan : sans les flags d'optimisation, Claude Code fait des appels supplémentaires pour les complétions, suggestions et télémétrie.
  • Long responses: responses containing generated code are often 5 to 20x longer than a regular chat.Reponses longues : les reponses contenant du code genere sont souvent 5 a 20x plus longues qu'un chat classique.

How to reduce consumptionComment reduire la consommation

1. Required environment variables1. Variables d'environnement obligatoires

# Réduit les appels en arrière-plan (complétions, suggestions)
DISABLE_NON_ESSENTIAL_MODEL_CALLS=1

# Désactive le trafic non-essentiel (télémétrie, checks)
CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1
Without these variables, consumption doubles or triplesSans ces variables, la consommation double ou triple

Claude Code makes dozens of additional calls that you don't see. These variables disable them.Claude Code fait des dizaines d'appels supplémentaires que vous ne voyez pas. Ces variables les desactivent.

2. Rate limiting on the proxy2. Rate limiting sur le proxy

# Minimum 30 secondes entre chaque requête
npx copilot-api@latest start --rate-limit 30 --wait

3. Disable web search3. Désactiver la recherche web

# Dans settings.json
"permissions": { "deny": ["WebSearch"] }

4. Choose economical models4. Choisir des modèles économes

ANTHROPIC_MODEL=gpt-5.4-mini          # au lieu de gpt-5.4
ANTHROPIC_SMALL_FAST_MODEL=gpt-5.4-mini

5. Usage best practices5. Bonnes pratiques d'utilisation

  • Be specific: "fix the bug on line 42 of server.js" consumes less than "fix the bugs"Soyez precis : "corrige le bug ligne 42 de server.js" consomme moins que "corrige les bugs"
  • Use /compact in Claude Code to compress contextUtilisez /compact dans Claude Code pour compresser le contexte
  • Close and restart Claude Code regularly to reset contextFermez et relancez Claude Code regulierement pour reinitialiser le contexte
  • Monitor quota: npx copilot-api@latest check-usageSurveillez le quota : npx copilot-api@latest check-usage
  • Avoid "agent loops" where Claude Code chains tools endlesslyÉvitez les "agent loops" ou Claude Code enchaine les outils sans fin
