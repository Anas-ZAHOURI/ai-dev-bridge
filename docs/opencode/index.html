What is OpenCode?C'est quoi OpenCode ?

OpenCode is an open source (MIT) terminal AI coding agent with 120K+ stars on GitHub. It is an alternative to Claude Code that supports 75+ providers including GitHub Copilot natively.OpenCode est un agent IA de coding en terminal, open source (MIT), avec 120K+ stars sur GitHub. C'est une alternative a Claude Code qui supporté 75+ providers dont GitHub Copilot nativement.

🌱
Open Source (MIT)
Open source code, no vendor lock-in, active community.Code source ouvert, pas de vendor lock-in, communauté active.
🔗
Native CopilotCopilot natif
Built-in GitHub Copilot support. No proxy, no hack.Support GitHub Copilot intégré. Pas de proxy, pas de hack.
⚙
75+ providers
OpenAI, Anthropic, Google, Ollama, Azure, Bedrock, and more.OpenAI, Anthropic, Google, Ollama, Azure, Bedrock, et plus.
💻
Modern TUITUI moderne
Bubble Tea terminal interface with vim keybindings.Interface terminal Bubble Tea avec keybindings vim.
Advantage vs Claude Code + proxyAvantage vs Claude Code + proxy

OpenCode supports GitHub Copilot natively. No need to install a proxy, no hack, no unofficial API. This is an official partnership between GitHub and OpenCode (January 2026).OpenCode supporté GitHub Copilot nativement. Pas besoin d'installer un proxy, pas de hack, pas d'API non-officielle. C'est un partenariat officiel entre GitHub et OpenCode (janvier 2026).

Copilot token consumptionConsommation de tokens Copilot

⚠ OpenCode consumes your Copilot quota⚠ OpenCode consomme votre quota Copilot

Every interaction with OpenCode consumes tokens from your GitHub Copilot license. A terminal agent consumes far more than a standard Copilot chat.Chaque interaction avec OpenCode consomme des tokens de votre licence GitHub Copilot. Un agent terminal consomme beaucoup plus qu'un chat Copilot classique.

Why consumption is highPourquoi la consommation est elevee

  • Large context: OpenCode sends file contents, history, and tool results with each request (10,000 - 50,000 input tokens)Contexte volumineux : OpenCode envoie le contenu des fichiers, l'historique, les resultats d'outils a chaque requête (10 000 - 50 000 tokens d'entrée)
  • Cascading calls: a single command can generate 3 to 15 API calls (read, reason, edit, verify)Appels en cascade : une commande peut generer 3 a 15 appels API (lecture, reflexion, edition, verification)
  • Long responses: responses containing code are 5 to 20x longer than a chatReponses longues : les reponses contenant du code sont 5 a 20x plus longues qu'un chat

Orders of magnitudeOrdres de grandeur

ActionEstimated tokensTokens estimesCopilot Chat equivalentEquivalent Copilot Chat
Simple questionQuestion simple5 000 - 15 0003-5 chat messages3-5 messages chat
File editEdition de fichier50 000 - 150 00030-50 chat messages30-50 messages chat
Complex taskTache complexe200 000 - 500 000100-200 chat messages100-200 messages chat
30 min sessionSession 30 min500 000 - 2 000 0001 full day of chat1 journee de chat

How to reduce consumptionComment reduire la consommation

  • Enable autoCompact (on by default): compresses context at 95% usageActivez autoCompact (actif par défaut) : compresse le contexte a 95% d'utilisation
  • Choose lightweight models: gpt-5-mini or gemini-3-flash for simple tasksChoisissez des modèles économes : gpt-5-mini ou gemini-3-flash pour les tâches simples
  • Be specific: "fix line 42 of server.js" uses fewer tokens than "fix the bugs"Soyez precis : "corrige la ligne 42 de server.js" consomme moins que "corrige les bugs"
  • Close and restart regularly to reset contextFermez et relancez regulierement pour reinitialiser le contexte
  • Non-interactive mode for one-off tasks: opencode -p "question"Mode non-interactif pour les tâches ponctuelles : opencode -p "question"
